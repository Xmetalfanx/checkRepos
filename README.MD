# Description 

This is a repo (ability to read user input is planned for later) to take a package name and search various distro repos for that package and display the exact version those repos have.

Distros:

- Arch
- Debian
    - Stable
    - Testing
    - Unstable
- Fedora
- OpenSUSE
    - Leap 15.5
    - Tumbleweed
- Ubuntu
    - Focal
    - Jammy
